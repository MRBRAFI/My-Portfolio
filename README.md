# Portfolio Website

This is a futuristic, highly interactive portfolio website designed to showcase projects, skills, and professional background. It features a modern "Iron Man" inspired aesthetic with cosmic themes, dynamic animations, and a responsive design.

## Features

*   **Dynamic Hero Section**: engaging introduction with visual flair.
*   **Interactive Header**: Floating navigation with a "CyberPet" companion and scroll progress indicators.
*   **About Section**: Detailed professional background.
*   **Projects Showcase**: A gallery of past work and projects.
*   **Skills Overview**: Visual representation of technical expertise.
*   **First Visit Loader**: A stunning, one-time loading animation sequence for new visitors.
*   **Glowing Borders**: Animated vertical border lines that add depth and movement to the page.
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
*   **Cosmic Theme**: Deep blue backgrounds, star fields, and glowing accents.

## Tech Stack

This project is built using the following technologies:

*   **Framework**: [React 19](https://react.dev/)
*   **Build Tool**: [Vite](https://vitejs.dev/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animations**:
    *   [Framer Motion](https://www.framer.com/motion/)
    *   [GSAP (GreenSock Animation Platform)](https://gsap.com/)
*   **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/MRBRAFI/My-Portfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd portfolio
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

Start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or another port if 5173 is in use).

### Building for Production

To create a production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Project Structure

*   `src/components`: Contains all the reusable React components (Header, Hero, About, Projects, etc.).
*   `src/assets`: Static assets like images.
*   `src/App.jsx`: The main application component that assembles the page.
*   `src/index.css`: Global styles and Tailwind directives.

## Components Breakdown

*   **FirstVisitLoader**: Handles the initial loading sequence.
*   **Header**: Contains the navigation bar and the interactive `CyberPet`.
*   **Hero**: The landing section of the portfolio.
*   **About**: Section detailing personal information.
*   **Projects**: Displays a grid of projects.
*   **Skills**: Lists technical skills.
*   **Footer**: Footer section with contact links.
*   **GlowingBorders**: Background animation adding vertical glowing lines.
*   **CyberPet**: An interactive element residing in the header.
